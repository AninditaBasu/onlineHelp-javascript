Twisties

Twisties are blocks of content that expand or collapse on click, and thus reveal or hide the information contained therein. Twisties are one method of making available additional information only on demand. In this example, the twistie is implemented through the twistie.js file by using a "twistie" class on the element that contains the additional information.

How to implement twisties

  • If you author in DITA, do these two things:
    1. Include the "twistie.js" and "twistie.css" files in your DITA to XHTML transformation. When a DITA element with an "outputclass" attribute is processed through DITA OT, the generated HTML element automatically gets a "class" attribute with the same value as the corresponding DITA element "outputclass".
    2. Put "twistie" as the value in the "outputclass" attribute of the element that contains the additional information, like this: 
      	<note outputclass = "twistie">Lorem lorem lorem<note>
  • If you author directly in HTML, do these two things:
    1. Include the following code within the "head" tag of the HTML file (and place the local files in the correct folders): 
      	<script src="https://code.jquery.com/jquery-1.11.3.min.js"></script>
	<script src = "https://code.jquery.com/ui/1.11.4/jquery-ui.js"></script>
	<script type="text/javascript" src="../scripts/twistie.js"></script>
	<link rel="stylesheet" type="text/css" href = "../css/twistie.css"/>
    2. For the element that contains the additional information, specify a "class" attribute with a value of "twistie", like this: 
      	<div class = "twistie">Lorem lorem lorem<div>

Where is the Javascript code and stylesheet file

The Javascript code is in the "scripts" folder, "twistie.js" file. The styling rule is in the "css" folder, "twistie.css" file.
