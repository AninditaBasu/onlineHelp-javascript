Documentation for the javascript snippets

The files use the jQuery and the jQuery UI libraries, and are implemented through the "class" attribute of HTML elements. The following table lists the javascript snippets available for your use, and the instructions for using them in your files.

Javascript file Effect When to use Instructions for use,
with working examples
twistie.js On click, show or hide blocks of content To provide additional information without otherwise breaking the flow of a bit of text in a topic twistie.html
fnotehovertext.js On mouseover, display the text of a footnote To give an option to the reader to not jump up and down within the document, but rather to stay within the context of the paragraph currently being read fnotehovertext.js
