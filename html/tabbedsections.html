Tabbed sections

Tabbed sections are one method of presenting the various sections of a page horizontally (instead of the default vertical method). Tabbed sections give the reader an immediate overview of the page content, and the facility to navigate between these blocks of content without needed to scroll up and down.

How to implement tabbed sections

  • If you author in DITA, ....
  • If you author in HTML, do these four things:
    1. Include the following code within the "head" tag of the HTML file (and place the local files in the correct folders): 
      	<script src="https://code.jquery.com/jquery-1.11.3.min.js"></script>
	<script src = "https://code.jquery.com/ui/1.11.4/jquery-ui.js"></script>
	<script type="text/javascript" src="../scripts/tabbedsections.js"></script>
	<link rel="stylesheet" type="text/css" href = "../css/tabbedsections.css"/>
    2. At the location where you want to place the tabs, create a block element such as <div>, and give it a "class" attribute with the value "tabs", like this: 
      	<div class = "tabs">...more elements...</div>
    3. Within the block element created in step 2, insert an <ul> element, where the list items are links to the several sections that will make up the tabs, like this: 
      	<div class = "tabs">
	<ul>
	<li><a href = "#tab1">Tab 1 heading</a></li>
	<li><a href = "#tab2">Tab 2 heading</a></li>
	<li><a href = "#tab3">Tab 3 heading</a></li>
	</ul>
	</div>
    4. At the point where the <ul> list gets over, insert all of those sections that will make up the tabs. Ensure that each section is one complete block element. Your code will look something like this: 
      	<div class = "tabs">
	<ul>
	<li><a href = "#tab1">Tab 1 heading</a></li>
	<li><a href = "#tab2">Tab 2 heading</a></li>
	<li><a href = "#tab3">Tab 3 heading</a></li>
	</ul>
	<div id = "tab1"><h3>Tab 1 heading</h3><p>....Tab 1 content....</p></div>
	<div id = "tab2"><h3>Tab 2 heading</h3><p>....Tab 2 content....</p></div>
	<div id = "tab3"><h3>Tab 3 heading</h3><p>....Tab 3 content....</p></div>
	</div>

Where is the Javascript code and stylesheet file

The Javascript code is in the "scripts" folder, "tabbedsections.js" file. The styling rules are in the "css" folder, "tabbedsections.css" file.
