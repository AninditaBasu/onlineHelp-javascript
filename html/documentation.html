Documentation for the javascript snippets

The files use the jQuery and the jQuery UI libraries, and are implemented through the "class" attribute of HTML elements. Which "class" attribute to use depends on the javascript effect that you want in your help files. The following table lists the javascript snippets available for your use, and the instructions for using them in your files.

File name Effect When to use Class to use Instructions for use
